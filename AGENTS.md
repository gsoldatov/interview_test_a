# Overview
A simple project implementing a web app integration with a relational DB and an Elastic full-text search index.

Stack used:
- Python 3.13 + uv;
- PostgreSQL 17;
- Elasticsearch 9;
- FastAPI;
- SQLAlchemy2 + psycopg3 + alembic;
- Pydantic + Pydantic Settings;
- pytest;
